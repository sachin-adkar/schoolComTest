# Parentheses balance & Simple user Signup/Signin

Requirements:
 - Node.js (12.3.0)
 - MongoDB (4.2.3)
 - yarn
 
# Stepts to start 
 Run the below commands in the terminal

     git clone git@github.com:sachin-adkar/schoolComTest.git
     yarn install
     yarn start
    
# Routes
 - /register : for signup
 - /login : for login
 - /users : list all the users
 - /validate : validate parenthesis
